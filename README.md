# Handwritten Digit Classifier                                                
                                                                                
A neural network that classifies handwritten digits (0-9) using the MNIST     
dataset.                                                                      
                                                                            
## What it does                                                               
- Trains a 3-layer neural network on 60,000 handwritten digit images
- Achieves ~97.6% accuracy on 10,000 unseen test images                       
- Visualises predictions alongside true labels                                
                                                                            
## Built with                                                                 
- Python                                                                      
- PyTorch                                                                     
- Matplotlib                                                                  

## How to run                                                                 
1. Clone the repository
2. Install dependencies: `pip install torch torchvision matplotlib`           
3. Run: `python digit_classifier.py`
